## Packages

In Go, every file begins with a `package` declaration, that it's a way of grouping all the files in the same directory under a given name. In this case we're using the name `main`, which tells the compiler that this file should be compiled as part of an **executable program**.

<Code code={mainHelloWorld} lang="go" title="main.go" mark={1} />

<Admonition type="important">
  Every **executable** Go program must have a `main` package.
</Admonition>

If we were to create a library instead, we would use a different package name.

## Importing Packages

Go has a rich standard library, and you can also import **third-party packages**. In our example, we imported the [fmt](https://pkg.go.dev/fmt) package, which provides functions for formatted I/O, including printing to the console.

## The main Function

The `main` function is the entry point of a Go program. When you run the program, execution starts from this function.
