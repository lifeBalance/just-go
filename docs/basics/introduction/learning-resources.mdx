# Learning Resources

Here are some resources to help you learn Go programming:


- [The Go Programming Language Book](https://www.gopl.io/): A comprehensive book by Alan A. A. Donovan and Brian W. Kernighan that covers Go in depth (also known as the **blue book**).
- [A Tour of Go](https://tour.golang.org/): An interactive online tutorial that introduces the basics of Go programming.
- [Go by Example](https://gobyexample.com/): A hands-on introduction to Go using annotated example programs.
- [Effective Go](https://go.dev/doc/effective_go): A guide to writing clear, idiomatic Go code.
- [Go Documentation](https://go.dev/doc/): The official Go documentation, including package references and tutorials.
- [Go Forum](https://forum.golangbridge.org/): A community forum for discussing Go programming and getting help.
- [Gophers Slack](https://gophers.slack.com/): A Slack community for Go developers to connect and collaborate.
- [Go Podcasts](https://changelog.com/gotime): Podcasts discussing Go programming and related topics.
- [Go YouTube Channel](https://www.youtube.com/c/golang): Official Go channel with talks, tutorials, and conference recordings.
- [Exercism Go Track](https://exercism.io/tracks/go): Practice Go programming with coding exercises and mentorship.

Some other cool sites:

- [Go Playground](https://play.golang.org/): An online environment to write, run, and share Go code snippets.
- [Go 101](https://go101.org/)
- [Gophersices](https://gophercises.com/), a **free** coding exercises site for learning Go.
- [Algorithms with Go](https://algorithmswithgo.com/): Another **free** resource.
- [Learn Go with Tests](https://quii.gitbook.io/learn-go-with-tests/): Learn Go using Test-Driven Development (TDD) approach.

## Web Development

- [Web Development with Go](https://www.usegolang.com/): From the author of **Gophersices** and **Algorithms with Go**.
