# Introduction

Go is a [statically typed](https://en.wikipedia.org/wiki/Type_system#Static_type_checking) language, which means that variables can only hold the type of data we specified during the declaration. We can categorize Go's [types](https://go.dev/ref/spec#Types) in four main groups:

* Basic types:
  * [Strings](https://golang.org/ref/spec#String_types)
  * [Numeric types](https://golang.org/ref/spec#Numeric_types)
  * [Booleans](https://golang.org/ref/spec#Boolean_types)

* Aggregate types
  * [Arrays](https://golang.org/ref/spec#Array_types)
  * [Structs](https://golang.org/ref/spec#Struct_types)

* Reference types
  * [Pointers](https://golang.org/ref/spec#Pointer_types)
  * [Slices](https://golang.org/ref/spec#Slice_types)
  * [Maps](https://golang.org/ref/spec#Map_types)
  * [Functions](https://golang.org/ref/spec#Function_types)
  * [Channels](https://golang.org/ref/spec#Channel_types)

* [Interfaces](https://golang.org/ref/spec#Interface_types)


## Booleans

The [boolean type](https://golang.org/ref/spec#Boolean_types) only contains two values denoted by the predeclared constants `true` and `false`. The [predeclared identifier](https://golang.org/ref/spec#Predeclared_identifiers) for the boolean type is `bool`.
