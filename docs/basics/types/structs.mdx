# Structs

A [struct](https://go.dev/ref/spec#Struct_types) is a sequence of named elements, called **fields**, each of which has a name and a type. Field names may be:

- Specified explicitly (IdentifierList). Field names must be unique.
- Implicitly (EmbeddedField).
