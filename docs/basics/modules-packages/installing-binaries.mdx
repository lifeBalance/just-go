# Installing Binaries vs Libraries

See: [`go install`](https://pkg.go.dev/cmd/go#hdr-Install_packages_and_dependencies) and [`go get`](https://pkg.go.dev/cmd/go#hdr-Add_dependencies_to_current_module_and_install_them).

- Install CLI tools: `go install <module>@<version>` (Go 1.17+)
- Add libraries: `go get` (updates `go.mod`), then import in code

```bash
# Binaries
go install github.com/cli/cli/cmd/gh@latest

# Libraries
go get github.com/acme/lib@v1.2.3
```

<Admonition variant="tip" title="Where binaries go?">
Installed binaries go to `$GOBIN` or `$GOPATH/bin`. Put it on your PATH.
</Admonition>
