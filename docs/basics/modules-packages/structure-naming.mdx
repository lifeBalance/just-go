# Structure & Naming

- Package names: short, lower-case; avoid stutter (`image.Image`).
- Layout:
  - `cmd/<app>`: binaries
  - `internal/`: non-exported packages for this module
  - `pkg/`: exported packages (optional; tastes vary)
